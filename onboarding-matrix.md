# Onboarding Entitlement Matrix

This matrix shows what each title receives at onboarding.

## Hardware

| Title | Laptop | Phone | Accessories |
|------|--------|-------|-------------|
| IT Support | Windows Laptop | iPhone | Dock, Headset |
| Sales Rep | MacBook | iPhone | Headset |
| Manager | MacBook | Optional | Dock |

## Software Access

| Title | Email | VPN | CRM | Admin Rights |
|------|-------|-----|-----|--------------|
| IT Support | Yes | Yes | No | Limited |
| Sales Rep | Yes | No | Yes | No |
| Manager | Yes | Yes | Yes | No |
